# 包内模块结构

> 此文件由 agents-docs-index skill 管理。

## 目录结构

- src/：源代码目录
  - index.ts：入口文件
  - test.ts：测试文件
- package.json：包配置文件
- tsconfig.json：TypeScript 配置文件
- .gitignore：Git 忽略文件
- README.md：包说明文件
- CLAUDE.md：Claude 相关文档

## 模块职责

- src/index.ts：包的主入口，导出主要功能
- src/test.ts：测试代码，用于验证功能
- package.json：包依赖和脚本配置
- tsconfig.json：TypeScript 编译配置
- README.md：包的说明和使用指南
- CLAUDE.md：Claude 相关的文档和说明

## 技术栈

- TypeScript
