# 包内规范

> 此文件由 agents-docs-index skill 管理。
> 以下规范覆盖根级 conventions

## 代码风格

- 遵循 Node.js 最佳实践
- 使用 TypeScript 进行类型检查
- 文件使用 camelCase 命名
- 函数和变量使用 camelCase 命名
- 常量使用 UPPER_SNAKE_CASE 命名

## 服务器规范

- 遵循 RESTful API 设计原则
- 错误处理统一
- 日志记录规范
- 环境变量配置

## 性能优化

- 合理使用中间件
- 数据库查询优化
- 缓存策略
- 并发处理

## 安全性

- 输入验证
- 防止 SQL 注入
- 防止 XSS 攻击
- 认证和授权

## 测试规范

- 为 API 端点编写测试
- 使用 Jest 进行单元测试
- 集成测试覆盖主要功能
